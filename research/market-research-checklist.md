**ユーザーの行動パターン**:
- たくさん紹介されて「おおぅ」ってなる

**本当に使いそうなショートカット（Instagramより）**:
- Ctrl + C（コピー）
- Ctrl + Z（元に戻す）

- Ctrl + S（保存）
- Ctrl + A（全選択）
- Ctrl + B（太字）
- Ctrl + F（検索）
- Ctrl + P（印刷）
- Ctrl + N（新規作成）

**重要な気づき**:

## 8. 市場調査まとめ
**目安時間: 10分**
- [x] 完了

### 市場の空白（チャンス）
1. **デザインが洗練されたものが少ない**（おじさんが作ったようなダサいデザインが多い）
2. **材質が良さそうなものが少ない**（安っぽい素材・質感）
3. **"できる人感"が出る洗練されたイメージのものが少ない**（かっこいい・スマート・センスがいい）
4. **ターゲット別に明確に分けられているものが少ない**（エンジニア向け・社会人向け・デザイナー向けなど）
5. **ブランド化されたものが少ない**（単発商品で終わっている）
6. **結局ターゲットが「情弱男子」や「おじさん」に絞られている**（若い女性・センスのいい人向けが少ない）
7. **Instagramで見て満足する層を商品化できていない**（保存して終わり→実際に手元に置きたい需要）

### 自分の差別化ポイント
1. **洗練されたデザインに仕上げられる**（デザインセンス・実行力がある）
2. **シンプルに本当に使えるものにできる**（厳選15〜20個・実用性重視）
3. **ブランド化まで持っていけるポテンシャルと気力がある**（やり遂げる力・継続力）
4. **周りの人に配ってレビューをもらえる環境がある**（宣材写真・フィードバック収集が可能）
5. **さまざまなターゲットに向けて展開できる**（22〜35歳男女・50代男女・小中学生など幅広い層）
6. **ターゲット別に明確に分ける戦略**（エンジニア・デザイナー・社会人・学生など）

### 懸念点・課題
1. **やり遂げられるか**（途中で挫折しないか・最後まで完走できるか）
2. **モチベーション維持**（継続できるか・面倒になって辞めないか）
3. **デザインの完成度**（想定より時間がかかりすぎないか・品質を保てるか）

### 商品設計の方針（Instagram調査より）
- **現実的な項目数**: 70×80mm～90×70mmのサイズで目に入って活用できるのは**15～20個が限界**
- **レイアウト**: ショートカット部分を**左側に配置**（右利き目が多いため、左から視線が入る）
- **厳選内容**:
  1. Ctrl系の超基本（C/V/Z/Y/S/A/F/H/P/N）
  2. スクリーンショット系
  3. アプリ切り替え・フォルダ操作の厳選ショートカット

### 次のアクション

**商品ライン戦略**:
1. **基本シール（Instagram情弱層向け）**: 超基本15〜20個・見た目重視・「デスクに置きたくなる」デザイン
2. **エンジニア向け（新規開拓）**: VSCode/ターミナル/Git厳選ショートカット

**具体的なタスク**:
- [ ] 15～20個に絞った「超基本ステッカー」のショートカットリストを作成

- [ ] グラフィック・StickerAppで見積もり取得
- [ ] メルカリでテスト販売準備

**マーケティング戦略**:
