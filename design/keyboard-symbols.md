# ⌨️ キーボード記号リファレンス

**最終更新: 2025/12/02**

このファイルは、ショートカットラベルのデザイン時に使用するキーボード記号の一覧です。
コピー&ペーストして使用してください。

---

## 🍎 Mac修飾キー

| キー名 | 記号 | Unicode | 説明 |
|--------|------|---------|------|
| **Command** | ⌘ | U+2318 | Macの主要な修飾キー |
| **Shift** | ⇧ | U+21E7 | シフトキー |
| **Option** | ⌥ | U+2325 | オプションキー（Alt） |
| **Control** | ⌃ | U+2303 | コントロールキー |
| **Caps Lock** | ⇪ | U+21EA | キャップスロック |

---

## ⌫ 削除キー

| キー名 | 記号 | Unicode | 説明 |
|--------|------|---------|------|
| **Delete (Backspace)** | ⌫ | U+232B | 左方向への削除（通常のDeleteキー） |
| **Forward Delete** | ⌦ | U+2326 | 右方向への削除（Fn + ⌫） |

---

## ➡️ 矢印キー

| キー名 | 記号 | Unicode | 説明 |
|--------|------|---------|------|
| **上** | ↑ | U+2191 | 上矢印 |
| **下** | ↓ | U+2193 | 下矢印 |
| **左** | ← | U+2190 | 左矢印 |
| **右** | → | U+2192 | 右矢印 |

---

## ⏎ 特殊キー

| キー名 | 記号 | Unicode | 説明 |
|--------|------|---------|------|
| **Return** | ⏎ | U+23CE | リターン/エンター |
| **Enter** | ⌅ | U+2305 | テンキーのエンター |
| **Tab** | ⇥ | U+21E5 | タブキー |
| **Escape** | ⎋ | U+238B | エスケープ |
| **Space** | ␣ | U+2423 | スペース（記号表示） |
| **Eject** | ⏏ | U+23CF | イジェクト |
| **Power** | ⌽ | U+233D | 電源 |

## 🔢 よく使う組み合わせ例

```
⌘ + C          コピー
⌘ + V          貼り付け
⌘ + ⌫          ゴミ箱へ移動
⌘ + ⇧ + 5      スクリーンショット
⌃ + ⌘ + Space  絵文字パネル
⌥ + ⌘ + Esc    強制終了
```

## 💡 使用上の注意

1. **デザインでの使用**: これらの記号はSF Pro, Inter, Poppinsなどのフォントでサポートされています
2. **代替表記**: 記号が表示されない場合は、テキスト表記を使用（例: Command, Shift）
3. **Affinity Designerでの入力**: 記号をコピー&ペーストするか、文字コード（Unicode）から入力可能
4. **視認性**: 大判印刷では記号の細かい線が潰れる可能性があるため、テストプリント推奨
