# Affinity Designerでステッカーデザインを作る手順

**最終更新: 2025/11/09**

---

## ◉ 事前準備

### 必要なもの
- [ ] Affinity Designer（インストール済み）
- [ ] ショートカットリスト（`/shortcut-lists/`フォルダ参照）
- [ ] 印刷業者の仕様確認（サイズ・解像度・入稿形式）

### デザインコンセプト
- **テイスト**: ミニマル・洗練された・プロっぽい（「できる人感」が出るデザイン）
- **色**: モノクロベース or 機能別カラー帯（視認性重視）
- **フォント**: 太めのゴシック体（視認性重視・パッと目に入る）
- **サイズ**: 70×80mm〜90×70mm（デスクに貼って目に入るサイズ）

## ◉ Step 1: 新規ドキュメント作成

### 1-1. Affinity Designerを起動

- 「ファイル」→「新規」をクリック


### 1-2. ドキュメント設定

| 項目 | 設定値 | 備考 |
|------|--------|------|
| **種類** | 印刷 | 印刷用途を選択 |
| **ページサイズ** | カスタム | - |
| **幅** | 90mm | 推奨サイズ |
| **高さ** | 70mm | 推奨サイズ |
| **DPI（解像度）** | 300dpi | 印刷品質を保つため |
| **カラーモード** | CMYK | 印刷用（RGBは画面表示用） |
| **裁ち落とし** | 3mm | 印刷業者の仕様に合わせる |


### 1-3. 保存

- 「ファイル」→「名前を付けて保存」
- ファイル名: `beginner-basic-shortcuts-v1.afdesign`
- 保存場所: `/design/` フォルダ

<br>
<br>

---

## ◉ Step 2: レイアウト設計

### 2-1. ガイドラインを引く

**目的**: デザインを整列させるための基準線を作成

1. 「表示」→「ガイド」→「ガイドマネージャー」を開く
2. 以下のガイドラインを追加:
   - **上下左右のマージン**: 5mm
   - **中央縦線**: 45mm（中央揃え用）
   - **ヘッダー下**: 15mm（タイトルエリア）


### 2-2. レイアウトゾーンを決める

| ゾーン | 位置 | 内容 |
|--------|------|------|
| **ヘッダー** | 上5mm〜15mm | タイトル「超基本ショートカット30選」 |
| **メインエリア** | 15mm〜65mm | ショートカット一覧（表形式） |
| **フッター** | 65mm〜70mm | ロゴ・ブランド名（オプション） |


### 2-3. グリッド設定（オプション）

- 「表示」→「グリッド」→「グリッドマネージャー」
- グリッド間隔: 5mm
- スナップを有効化

<br>
<br>

---

## ◉ Step 3: タイポグラフィ（文字設定）

### 3-1. フォント選定

**おすすめフォント（視認性重視）**:

| 用途 | フォント候補 | 太さ |
|------|------------|------|
| **タイトル** | Noto Sans JP Bold / Hiragino Sans W6 | 太字 |
| **ショートカット（キー部分）** | Noto Sans JP Bold / SF Pro Display Bold | 太字 |
| **説明文** | Noto Sans JP Regular / Hiragino Sans W3 | 中字 |


### 3-2. 文字サイズ設定

| 箇所 | サイズ | 用途 |
|------|--------|------|
| **タイトル** | 18〜20pt | ヘッダー |
| **ショートカットキー** | 12〜14pt | メイン（Ctrl + C など） |
| **説明文** | 8〜10pt | サブテキスト |


### 3-3. テキストツールで入力

1. **テキストツール（T）** を選択
2. タイトルを入力: 「超基本ショートカット30選」
3. フォント・サイズ・色を設定
4. 中央揃えに配置

<br>
<br>

---

## ◉ Step 4: ショートカット一覧のデザイン

### 4-1. 表形式レイアウト

**レイアウト案（左配置・視認性重視）**:

```
┌────────────────────────────────────┐
│  Ctrl + C     コピー               │
│  Ctrl + V     貼り付け             │
│  Ctrl + Z     元に戻す             │
│  Ctrl + S     保存                 │
│  ...                               │
└────────────────────────────────────┘
```

**ポイント**:
- ショートカット（左側）: 太字・大きめ
- 説明（右側）: 通常フォント・小さめ
- 行間: 適度に空ける（詰まりすぎない）


### 4-2. 長方形ツールで背景を作成

1. **長方形ツール（M）** を選択
2. ショートカット1行分の背景を作成
3. 色: 白 or 薄いグレー（交互に配置で見やすく）
4. 行間: 2〜3mm


### 4-3. テキストを配置

1. **テキストツール（T）** で各ショートカットを入力
2. **ショートカット部分（Ctrl + C）**: 左揃え・太字
3. **説明部分（コピー）**: 右揃え or 左揃え・通常フォント
4. **整列ツール** で位置を調整


### 4-4. 色分け（オプション）

**カテゴリ別に色分けする場合**:

| カテゴリ | 色 | 用途 |
|---------|---|------|
| **Windows基本** | 青系（#2E86AB） | Windows操作 |
| **Mac基本** | グレー系（#6C757D） | Mac操作 |
| **Excel基本** | 緑系（#06A77D） | Excel操作 |

## ◉ Step 5: デザイン調整

### 5-1. 視認性チェック

- [ ] **文字が小さすぎないか**（最小8pt以上推奨）
- [ ] **行間が詰まりすぎていないか**（適度な余白）
- [ ] **ショートカットが左側に配置されているか**（視線の流れ）
- [ ] **色のコントラストが十分か**（背景と文字の差）


### 5-2. 余白チェック

- [ ] **上下左右のマージン**: 最低5mm確保
- [ ] **裁ち落とし**: 3mm設定済み
- [ ] **重要な情報が裁ち落としエリアに入っていないか**


### 5-3. 整列・配置

- 「整列パネル」を使用して、全要素を揃える
- 中央揃え・左揃えを統一

<br>
<br>

## ◉ Step 6: エクスポート（入稿データ作成）

### 6-1. 印刷業者の仕様確認

**主要印刷業者の入稿形式**:

| 業者 | 入稿形式 | 備考 |
|------|---------|------|
| **グラフィック** | PDF / AI / PSD | CMYKモード・裁ち落とし3mm |
| **StickerApp** | AI / PDF / PNG | 高解像度PNG可 |
| **プリントパック** | PDF / AI | PDF推奨 |


### 6-2. PDF形式でエクスポート

1. 「ファイル」→「エクスポート」→「PDF」を選択
2. 設定:
   - **PDF形式**: PDF/X-1a:2001（印刷用標準）
   - **カラーモード**: CMYK
   - **解像度**: 300dpi
   - **裁ち落とし**: 3mm
   - **トンボ**: 含める（印刷業者による）
3. ファイル名: `beginner-basic-shortcuts-v1.pdf`
4. 保存場所: `/design/export/` フォルダ


### 6-3. AI形式でエクスポート（オプション）

1. 「ファイル」→「エクスポート」→「SVG」を選択
2. その後Illustratorで開いてAI形式で保存
   - または「ファイル」→「エクスポート」→「AI（Illustrator）」

<br>
<br>

## ◉ Step 7: 印刷テスト

### 7-1. 自宅プリンターでテスト印刷

- 実際のサイズ（90×70mm）で印刷
- 文字の視認性を確認
- 色味を確認（モニターと印刷では異なる）


### 7-2. チェック項目

- [ ] **文字が読めるか**（小さすぎないか）
- [ ] **色が想定通りか**（CMYKとRGBの差）
- [ ] **余白が適切か**（詰まりすぎていないか）
- [ ] **裁ち落としが正しいか**


### 7-3. 修正・再エクスポート

- 問題があれば修正
- バージョン番号を上げて保存（v2, v3...）

## ◉ 次のステップ

1. **デザイン完成** → `/design/export/` に保存
2. **印刷業者に見積もり依頼** → グラフィック・StickerApp
3. **友人にテスト配布** → フィードバック収集
4. **デザイン修正** → v2, v3と改善
5. **本番印刷発注** → 少数ロット（10〜30枚）
6. **販売開始** → メルカリ・BOOTHでテスト販売
