## ⚙️ Affinity Designerでの一括設定手順

### フォントリソース管理（不足フォントの警告・置換）

**目的**: デザインファイルで使用しているフォントが不足している場合、警告を表示して置換できる。

#### フォントリソース管理の使い方

1. 「テキスト」→「フォントリソース管理」をクリック
2. 不足しているフォントがある場合、リストに表示される
3. 代替フォントを選択して一括置換


#### 使用例

- SF Pro Displayが入っていない場合 → Interに一括置換
- 他の人に共有するときに、フォントがない場合の対処

### テキストスタイル機能（一括設定）

**目的**: タイトルや文字のスタイルを作成・保存・インポートし、ドキュメント全体に一括で適用できる。

#### テキストスタイルの種類

- **段落スタイル**: 段落全体の書式（フォント・サイズ・行間・配置など）
- **キャラクタースタイル**: 文字単位の書式（フォント・サイズ・色など）


#### Step 1: テキストスタイルパネルを開く

**方法A（メニューから）**:
1. 「表示」→「スタジオ」→「テキストスタイル」
2. または「ウィンドウ」→「テキストスタイル」

**方法B（キーボードショートカット）**:
- Windows: F5キーを押す
- Mac: F5キーを押す

**パネルが見つからない場合**:
- 画面右側の縦長のアイコンバーから「A」のようなアイコンをクリック


#### Step 2: テキストスタイルを作成

**1. Titleスタイル（見出し用・段落スタイル）**

1. テキストツール（T）でテキストを入力
2. フォント設定:
   - フォント: **SF Pro Display Bold**
   - サイズ: 100pt
   - カラー: #FFFFFF（白）
   - 配置: 中央揃え
3. テキストを選択した状態で「テキストスタイル」パネルの「+」ボタンをクリック
4. 「段落スタイル」を選択
5. スタイル名: 「Title」→「作成」


**2. Commandスタイル（操作名用・段落スタイル）**

1. テキストを入力してフォント設定:
   - フォント: **SF Pro Display Medium**
   - サイズ: 60pt
   - カラー: #FFFFFF（白）
2. 「テキストスタイル」パネルで「+」→「段落スタイル」
3. スタイル名: 「Command」


**3. Descriptionスタイル（説明文用・キャラクタースタイル）**

1. テキストを入力してフォント設定:
   - フォント: **SF Pro Text Regular**
   - サイズ: 45pt
   - カラー: #EAEAEA（灰色）
2. 「テキストスタイル」パネルで「+」→「キャラクタースタイル」
3. スタイル名: 「Description」


**4. KeyLabelスタイル（キー文字用・キャラクタースタイル）**

1. テキストを入力してフォント設定:
   - フォント: **SF Pro Rounded Bold**
   - サイズ: 70pt
   - カラー: #1C1C1E（黒）
2. 「テキストスタイル」パネルで「+」→「キャラクタースタイル」
3. スタイル名: 「KeyLabel」


#### Step 3: テキストスタイルを適用

1. スタイルを適用したいテキストを選択
2. 「テキストスタイル」パネルでスタイル名をクリック
3. **ドキュメント全体に一括適用完了！**


#### Step 4: テキストスタイルを編集（一括変更）

1. 「テキストスタイル」パネルでスタイル名を右クリック
2. 「編集」を選択
3. フォント・サイズ・色を変更
4. 「更新」をクリック
5. **全ての同じスタイルのテキストが一括で変更される！**


#### Step 5: テキストスタイルの保存・インポート

**保存**:
1. 「テキストスタイル」パネルのメニュー（右上の3点アイコン）
2. 「エクスポート」を選択
3. ファイル名をつけて保存（.afstyles）

**インポート**:
1. 「テキストスタイル」パネルのメニュー
2. 「インポート」を選択
3. 保存したスタイルファイルを読み込み
4. **別のドキュメントでも同じスタイルを使える！**

### カラーの一括設定（スウォッチパネル）

**目的**: 色を毎回HEXで入力するのは面倒。スウォッチパネルで色の追加・保存・プリセットカラーの適用が可能。

#### Step 1: スウォッチパネルを開く

**方法A（メニューから）**:
1. 「表示」→「スタジオ」→「スウォッチ」
2. または「ウィンドウ」→「スウォッチ」

**方法B（キーボードショートカット）**:
- Windows: F9キーを押す
- Mac: F9キーを押す

**パネルが見つからない場合**:
- 画面右側の縦長のアイコンバーから「カラーパレット」アイコンをクリック


#### Step 2: カラーをスウォッチに追加

**1. 背景色（#1C1C1E）を追加**

1. カラーパネルで「#1C1C1E」を入力
2. スウォッチパネルの「+」ボタン（またはドラッグ）で追加
3. 右クリック→「名前を変更」→「背景」


**2. 枠線（#2C2C2E）を追加**

1. カラーパネルで「#2C2C2E」を入力
2. スウォッチパネルに追加
3. 名前: 「枠線」


**3. キー背景（#FAFAFA）を追加**

1. カラーパネルで「#FAFAFA」を入力
2. スウォッチパネルに追加
3. 名前: 「キー背景」


**4. テキスト白（#FFFFFF）を追加**

1. カラーパネルで「#FFFFFF」を入力
2. スウォッチパネルに追加
3. 名前: 「テキスト白」


**5. テキスト灰（#EAEAEA）を追加**

1. カラーパネルで「#EAEAEA」を入力
2. スウォッチパネルに追加
3. 名前: 「テキスト灰」


#### Step 3: スウォッチから色を適用

1. オブジェクトを選択
2. スウォッチパネルで保存した色をクリック
3. 一瞬で適用完了！


#### Step 4: プリセットカラーを活用

- スウォッチパネルには予めプリセットカラーが用意されている
- 必要に応じてプリセットを使用してデザインを効率化

### シンボル機能でキーを再利用（効率化）

**目的**: キーのデザインを1つ作って、30個全てに適用。修正も一括反映。

#### Step 1: キーのデザインを作成

1. 長方形ツール（M）でキーの背景を作成
   - サイズ: 適当（後で調整可能）
   - 塗り: #FAFAFA（キー背景）
   - 線: #2C2C2E（枠線）
   - 角丸: 5px（オプション）

2. テキストツール（T）でキー文字を入力
   - 例: 「Ctrl」
   - スタイル: KeyLabel

3. 長方形とテキストをグループ化
   - 両方を選択 → 「Ctrl + G」（Mac: Command + G）


#### Step 2: シンボルに変換

1. グループ化したキーを選択
2. 「シンボル」パネルで「シンボルを作成」をクリック
3. 名前: 「Key」


#### Step 3: シンボルをコピー

1. シンボルパネルから「Key」をドラッグ&ドロップ
2. 30個配置
3. テキスト部分だけ編集（Ctrl → V → X など）


#### Step 4: シンボルを一括編集

1. シンボルパネルで「Key」をダブルクリック
2. デザインを編集（色・サイズ・角丸など）
3. 編集完了後、元のドキュメントに戻る
4. **全てのキーが一括で変更される！**

## 📐 レイアウト設計（800×800mm）

### ドキュメント設定

| 項目 | 設定値 |
|------|--------|
| **幅** | 800mm |
| **高さ** | 800mm |
| **DPI（解像度）** | 300dpi |
| **カラーモード** | CMYK |
| **裁ち落とし** | 3mm |


### エリア分割

```
┌─────────────────────────────────────┐
│  ヘッダー（タイトル）: 80mm         │
├─────────────────────────────────────┤
│  1行目（15個）: 340mm                │
├─────────────────────────────────────┤
│  2行目（15個）: 340mm                │
├─────────────────────────────────────┤
│  フッター（ブランド名など）: 40mm   │
└─────────────────────────────────────┘
```

### 1行あたりのサイズ

- **横幅**: 800mm - 左右マージン30mm = 770mm
- **1項目あたりの幅**: 770mm ÷ 15個 = 51mm
- **縦幅**: 340mm ÷ 15個 = 22.7mm（1項目あたり）


### レイアウト例（15個横並び）

```
┌─────────┬─────────┬─────────┬─────────┬─────────┐
│ Ctrl+C  │ Ctrl+V  │ Ctrl+X  │ Ctrl+Z  │ Ctrl+S  │ ...
│         │         │         │         │         │
│ コピー  │ 貼付け  │ 切取り  │ 戻す    │ 保存    │
└─────────┴─────────┴─────────┴─────────┴─────────┘
```

**ポイント**:
- 51mm幅に収まるように
- 上段: ショートカットキー（70pt・太字）
- 下段: 説明（50pt・通常）
- 余白: 各セル間3mm

## 🖨️ 印刷について

### 800×800mmの印刷

- かなり大きいサイズなので、印刷業者に要確認
- グラフィック・StickerAppで対応可能か見積もり依頼
- 分割印刷（400×400mm × 4枚）も検討
- ロール紙印刷に対応している業者を探す

### コスト

- 大判印刷はコスト高め（1枚1000円〜3000円程度）
- 少数ロットは割高になる可能性

### 実物サイズ感

**800×800mm = 80cm × 80cm**

- A1サイズ（594×841mm）より少し小さい
- デスクの壁に貼るサイズ
- かなり大きい！遠くからでも見える
- 「やってる感」「プロ感」が出る
