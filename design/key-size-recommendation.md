## 📐 レイアウト計算

### 基本設定
- **全体サイズ**: 800×800mm
- **配置**: 15個×2行（合計30個）
- **余白**: 上下左右30mm

## 🔑 キーサイズの推奨

### 計算

**横幅**:
- 使える幅: 800mm - 左右マージン30mm × 2 = 740mm
- 15個配置する場合: 740mm ÷ 15 = 49.3mm

**縦幅（1行あたり）**:
- 1行の高さ: 340mm
- ショートカット部分 + 説明部分を考慮


### パターン1: 正方形型（推奨）

**キー1個あたりのサイズ**:
- **横幅**: 45mm
- **縦幅**: 70mm（キー部分45mm + 説明部分25mm）
- **キー間の余白**: 4mm

**内訳**:
```
┌─────────────────┐
│   ┌─────────┐   │
│   │  Ctrl   │   │ ← キー本体 45×45mm
│   │    +    │   │    （正方形・角丸10px）
│   │    C    │   │    キーボードっぽい！
│   └─────────┘   │
│                 │
│   コピー         │ ← 説明文エリア 25mm
│                 │
└─────────────────┘
   45mm × 70mm
```

**特徴**:
- 正方形でキーボードっぽい
- 間延びしない・締まって見える
- キーが大きくて見やすい
- 説明文も読みやすい

## 🎨 デザインの詳細

### キー本体のデザイン

**色**:
- 背景: #FAFAFA（薄い白）
- 枠線: #2C2C2E（ダークグレー）2px
- 影: なし or 軽いドロップシャドウ

**角丸**:
- 推奨: 10px
- キーボードっぽさが出る

**キー文字**:
- フォント: Poppins Bold
- サイズ: 60〜70pt
- カラー: #1C1C1E（黒）
- 配置: 中央揃え


### 説明文のデザイン

**説明文**:
- フォント: Inter Regular
- サイズ: 40〜50pt
- カラー: #EAEAEA（薄い灰色）
- 配置: 中央揃え
- 行間: 適度に空ける


### レイアウトのポイント

1. **キーと説明の距離**: 5〜10mm空ける
2. **キー間の余白**: 4mm（詰まりすぎない）
3. **視認性**: 1メートル離れても読める
4. **統一感**: 全てのキーを同じサイズに

## 📝 Affinity Designerでの設定

### キー本体を作成

1. **長方形ツール（M）**
2. サイズ: 45mm × 40mm
3. 塗り: #FAFAFA
4. 線: #2C2C2E（2px）
5. 角丸: 10px（コーナーツールで調整）


### キー文字を配置

1. **テキストツール（T）**
2. フォント: Poppins Bold / 70pt
3. カラー: #1C1C1E
4. テキスト: 「Ctrl + C」など
5. 中央揃えで配置


### 説明文を配置

1. **テキストツール（T）**
2. フォント: Inter Regular / 45pt
3. カラー: #EAEAEA
4. テキスト: 「コピー」など
5. キーの下に配置（余白5mm）


### シンボル化して再利用

1. キー全体（長方形+文字）をグループ化
2. シンボルパネルで「シンボルを作成」
3. 名前: 「Key」
4. 30個配置して、文字だけ変更
