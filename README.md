## ◉ アイデアの背景

### なぜこれをやりたいか

- 既存のショートカットキーステッカーは「デザインが微妙・品質が低い」ものが多い
- 実用的で美しいデザインのステッカーを作ることで、PCユーザーの作業効率と満足度を両立できる
- 自分でデザインから販売まで完結させることで、コスト削減とデザイン品質向上を実現
- 「学び × デザイン × 実用性」を兼ね備えたプロダクトを形にしたい


### 自分の興味・強み・得意

- デザインセンス（フォント・配色・レイアウト設計）
- やり遂げる力（企画→デザイン→販売まで完走できる）
- 実用性重視（よく使うショートカットを体系的に整理できる）
- マーケティング感性（ユーザーが本当に欲しいものを形にする）


### 需要があるか（誰が必要としているか）

- **エンジニア・プログラマー**: キーボード操作を効率化したい層
- **デザイナー・クリエイター**: Adobe製品やデザインツールのショートカットを覚えたい層
- **新社会人・学生**: PCスキルを向上させたい初心者層
- **作業効率を重視する人**: Mac/Windowsユーザー全般


**使用シーン**:

- PCのキーボード周辺に貼ってショートカットを覚える
- デスクに貼って作業効率を可視化
- 見た目もおしゃれで、デスク環境の一部として楽しめる

## ◉ MVP（最小実行形）

### 今月中に「1円でも動く」状態の定義

- デザイン完成・印刷業者決定・販売ページ作成・実物テスト販売スタート
- 友人への試供品配布と感想収集
- メルカリまたはBOOTHでの初回販売開始


### 必要な準備・作業

**1. 市場調査**

- 競合商品のリサーチ（メルカリ・BOOTH・Amazon・Etsy）
- ターゲット層のニーズ調査（SNS・デスク周り投稿分析）
- 価格設定とデザイン傾向の分析
- 詳細は [市場調査チェックリスト](./research/market-research-checklist.md) を参照


**2. デザイン作成**

- Illustrator/Figmaでデザイン作成（フォント・配色・サイズ統一）
- よく使うショートカットキーを整理（Mac／Windows別・共通）
- ショートカットキーをカテゴリごとにまとめる（作業効率／検索／編集／開発）


**3. 印刷業者選定**

- グラフィックで紙質・質感を比較（マット・光沢・透明など）
- 少数ロットで試作印刷を依頼


**4. 販売媒体の決定**

- メルカリ or BOOTH（匿名発送・低コスト）を選定・登録
- 必要に応じてBASE／Shopifyも検討


**5. 初回テスト販売**

- 撮影・商品ページ作成
- 友人テスト販売


### 必要なツール・サービス

- **デザイン**: Illustrator / Figma（ステッカーのレイアウト・配色設計）
- **印刷**: グラフィック or StickerApp（小ロット印刷・紙質比較）
- **販売**: メルカリ / BOOTH / BASE（テスト販売・レビュー収集）
- **SNS**: Instagram / X（デスク写真・制作過程の発信）

## ◉ リスク・課題

### 予想される障害

1. **継続の難しさ**: 面倒になって挫折する
2. **販売停滞**: 販売チャネルの弱さで売れない
3. **デザインの完成度**: 想定より時間がかかる
4. **コスト**: 初期投資（印刷費・デザインツール費）が回収できない


### それにどう対処するか

1. **継続の難しさ対策**: スケジュールを細分化・1週間単位で進捗確認
2. **販売停滞対策**: SNS連携・口コミ・レビュー獲得を積極的に行う
3. **デザインの完成度対策**: 試作を繰り返し、ユーザーフィードバックを反映
4. **コスト対策**: 少数ロットから開始し、需要を見ながら段階的に拡大
