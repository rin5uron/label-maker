## ◉ プロジェクト概要

### コンセプト

既存のショートカットキーステッカーは「デザインが微妙・品質が低い」ものが多い。実用的で美しいデザインのステッカーを作ることで、PCユーザーの作業効率と満足度を両立する。

### ターゲット

- **エンジニア・プログラマー**: キーボード操作を効率化したい層
- **デザイナー・クリエイター**: デザインツールのショートカットを覚えたい層
- **新社会人・学生**: PCスキルを向上させたい初心者層
- **作業効率を重視する人**: Mac/Windowsユーザー全般

### 使用シーン

- PCのキーボード周辺に貼ってショートカットを覚える
- デスクに貼って作業効率を可視化
- 見た目もおしゃれで、デスク環境の一部として楽しめる
