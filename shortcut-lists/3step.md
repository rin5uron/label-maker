# 🎓 初級編ショートカット（Mac基準・具体的用途1行）

| No. | ショートカット | 意味 | 具体的用途（自然文で1行） |
|----|----------------|------|----------------|
| 1 | ⌘ + C | コピー | 選択した文字や画像をコピー |
| 2 | ⌘ + V | 貼り付け | ⌘Cでコピーした内容を貼り付け |
| 3 | ⌘ + X | 切り取り | 選んだ内容を削除し、他の場所に⌘Vで貼り付け |
| 4 | ⌘ + Z | 元に戻す | 直前の操作を元に戻す|
| 5 | ⇧ + ⌘ + Z | やり直す | 戻しすぎた時にもう一度進める |
| 6 | ⌘ + S | 保存 | こまめな保存に最適、突然のエラー対策にも |
| 7 | ⌘ + A | すべて選択 | 一括で全てを選択できる、一括削除やコピーに便利 |
| 8 | ⌘ + T | 新しいタブを開く | ブラウザやfinderで新しいタブを開く |
| 9 | ⌘ + W | タブを閉じる | ブラウザやエディタのタブをすぐ閉じられ整理しやすい |
| 10 | ⌘ + Q | アプリを終了 | アプリ全体を完全に閉じて動作を軽くする |
| 11 | ⌘ + F | 検索 | ページや文書内で特定の単語を探す、ファイル名の検索 |
| 12 | ⌘ + P | 印刷 | 印刷設定画面にすぐ飛べる |
| 13 | ⌘ + ⇧ + 4 | 範囲スクリーンショット | 必要な部分だけを画像で切り取る 、全画面スクリーンショットは⌘ + ⇧ + 3|
| 14 | スペースキー（選択中） | クイックルック | Finderでファイルを開かずに中身を確認できる。 |
| 15 | ⌘ + Tab | アプリ切り替え | 複数アプリを素早く行き来できる。作業の切り替えに◎。 |


# ⚙️ 中級編ショートカット（Mac基準・具体的用途1行）

| No. | ショートカット | 意味 | 具体的用途（自然文で1行） |
|----|----------------|------|----------------|
| 1 | ⌘ + ⇧ + 3 | 全画面スクリーンショット | 画面全体を画像で保存。資料づくりに便利。 |
| 2 | ⌘ + ⇧ + 5 | 画面収録メニューを開く | 必要な部分の録画やキャプチャがすぐできる。 |
| 3 | ⌘ + N | 新しいウィンドウを開く | Finderやブラウザで新しい画面を開ける。 |
| 4 | ⌘ + H | ウィンドウを隠す | 今のアプリを一時的に隠して画面をすっきり。 |
| 5 | ⌘ + M | ウィンドウを最小化 | 作業中のウィンドウをDockにしまえる。 |
| 6 | ⌘ + ,（カンマ） | 設定を開く | アプリごとの環境設定をすぐ開ける。 |
| 7 | ⌘ + ⇧ + N | 新しいフォルダを作成 | Finderで整理フォルダをすぐ作れる。 |
| 8 | ⌘ + Delete | 選択項目を削除 | ファイルをゴミ箱にすぐ移動できる。 |
| 9 | ⌘ + ⇧ + Delete | ゴミ箱を空にする | 一気に不要ファイルを削除して軽くできる。 |
| 10 | ⌘ + L | アドレスバーを選択 | ブラウザでURLをすぐ入力できる。検索が早い。 |
| 11 | ⌘ + ↑ / ↓ | ページの先頭・末尾へ移動 | 長文を一気にスクロールできる。 |
| 12 | Option + ← / → | 単語単位でカーソル移動 | 文章修正がスムーズになる。 |
| 13 | ⌘ + Option + Esc | 強制終了 | フリーズしたアプリを強制的に閉じられる。 |
| 14 | ⌘ + Space | Spotlight検索を開く | アプリ・ファイルを一瞬で探せる。 |
| 15 | Control + Space | 入力言語を切り替え | 英字↔日本語入力をサッと変えられる。 |


# 🧠 上級編ショートカット（Mac基準・具体的用途1行）

| No. | ショートカット | 意味 | 具体的用途（自然文で1行） |
|----|----------------|------|----------------|
| 1 | ⌘ + ⇧ + G | パス指定で移動 | Finderで特定フォルダに一発アクセスできる。 |
| 2 | ⌘ + Option + T | ツールバーの表示切替 | 画面を広く使いたい時にツールバーを非表示にできる。 |
| 3 | ⌘ + Option + H | 他のアプリをすべて隠す | 作業中のアプリだけ残して集中できる。 |
| 4 | Control + Tab | タブ間を順に切り替える | デザインやコードで複数ファイルを行き来できる。 |
| 5 | Control + Shift + Tab | タブを逆順に切り替える | 前のタブに戻りたい時に使える。 |
| 6 | Option + Delete | 単語単位で削除 | 文章修正が圧倒的に速くなる。 |
| 7 | ⌘ + D | 選択範囲を複製 | デザインソフトやテキストで同じ要素をすぐ増やせる。 |
| 8 | ⌘ + ⇧ + D | 最近使ったフォルダを開く | Finderで前回の作業フォルダにすぐ戻れる。 |
| 9 | ⌘ + Option + I | 情報ウィンドウを開く | ファイルサイズや詳細をすぐ確認できる。 |
| 10 | ⌘ + ⇧ + .（ドット） | 隠しファイルの表示切替 | システムファイルを一時的に見たい時に。 |
| 11 | ⌘ + Option + Control + Power | 再起動 | フリーズ時に強制リセットできる。 |
| 12 | ⌘ + Option + Eject | すべてのディスプレイをスリープ | 作業中でもすぐ画面ロックできる。 |
| 13 | Fn + ↑ / ↓ | ページ単位でスクロール | 長文やコードを一気に読むときに便利。 |
| 14 | Control + Command + Q | 画面ロック | 席を離れる時にすぐロックできる。 |
| 15 | ⌘ + Option + Esc | アプリを強制終了 | 固まったアプリを強制的に終了できる。 |
