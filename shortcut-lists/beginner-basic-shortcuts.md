# 初心者向け 超基本ショートカット30選

**ターゲット**: 新社会人・学生・PC初心者
**目的**: 絶対に覚えておきたい基本中の基本

---

## ◉ Windows基本操作（15個）

| No. | ショートカット | 説明 | 使用場面 |
|-----|--------------|------|---------|
| 1 | Ctrl + C | コピー | テキストやファイルをコピー |
| 2 | Ctrl + V | 貼り付け | コピーした内容を貼り付け |
| 3 | Ctrl + X | 切り取り | テキストやファイルを移動 |
| 4 | Ctrl + Z | 元に戻す | 操作を1つ前に戻す |
| 5 | Ctrl + Y | やり直し | 元に戻した操作をやり直す |
| 6 | Ctrl + S | 保存 | ファイルを保存 |
| 7 | Ctrl + A | 全選択 | 全てのテキストやファイルを選択 |
| 8 | Ctrl + F | 検索 | ページ内やファイル内を検索 |
| 9 | Ctrl + P | 印刷 | 印刷ダイアログを開く |
| 10 | Ctrl + N | 新規作成 | 新しいファイルやウィンドウを開く |
| 11 | Ctrl + W | ウィンドウを閉じる | 現在のウィンドウやタブを閉じる |
| 12 | Alt + Tab | アプリ切り替え | 開いているアプリを切り替え |
| 13 | Windows + D | デスクトップ表示 | 全てのウィンドウを最小化 |
| 14 | Windows + L | ロック | PCをロック画面にする |
| 15 | Windows + E | エクスプローラー | ファイル管理画面を開く |

- **文字サイズ**: 大きく・太く（視認性重視）

- **色分け**: Windows（青）/ Mac（グレー）/ Excel（緑）で区別
- **サイズ**: 70×80mm〜90×70mm
